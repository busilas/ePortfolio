02 MODULE

Object Oriented Programming.

This module is like learning to play the guitar. Each unit is a string on the guitar, contributing to a harmonious whole when played together. I have learned the "notes" of object-oriented programming (OOP) using Python and how to compose these notes into a beautiful melody of well-structured software.

  01 UNIT

An Introduction to Python Programming and the OO Programming Paradigm

Required Reading

In this first unit, I "tuned my strings" by understanding the basics of Python and OOP fundamentals. Just as a guitarist must tune their strings before playing, I started with the essential concepts of OOP like abstraction, encapsulation, inheritance, and polymorphism. These principles were the foundation of writing effective object-oriented programs. I also delved into defining Python classes and different data types.

By the end of this unit, I was able to describe the major features of an object-oriented program, define a Python class, and assign various data types to variables in a Python program. This was my starting point, ensuring we had a solid understanding before moving forward.
